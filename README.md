# Intelcomp Expert

**Professional Vocal Dynamics Engine** with intelligent, context-aware processing.

## Features

- **6-Band Multiband Compression** with vocal-specific crossover points
- **Lookahead Processing** for transparent transient control
- **Soft Knee Compression** for musical, natural sound
- **Dedicated Sibilance Detector** with spectral flux analysis
- **Transient Preservation** to maintain punch and articulation
- **MFCC + Spectral Centroid Intelligence** for timbre-aware processing
- **Parallel Compression with Saturation** for analog-style warmth
- **ITU-R BS.1770 LUFS Metering** for broadcast/streaming compliance
- **True Peak Limiting** for safe delivery
- **5 Professional Presets** (Pop, Rock, Ballad, Rap, Podcast)
- **Modern GUI** with waveform visualization and A/B comparison
- **CLI Mode** for batch processing
- **YAML Configuration** for custom settings
- **Numba Acceleration** for fast processing
